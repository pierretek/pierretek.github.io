ESP32-Hosted Webserver

I Built an ESP32 based webserver, which can recieve commands to control an array of four LEDs.

The webserver hosts a terminal type interface where the user can control individual or multiple LEDs to turn on or blink for a specified amount of time

The web interface is made with HTML, CSS, and JavaScript while the ESP32 is programmed with C++

Simple VHDL Processor

Architecture for an simple general purpose processsor written in VHDL

This architecture recieves two 8-bit values and performs specific functions depending on the provided opcode. The processor was waveform tested then implemented onto a real FPGA board

The input, output, and opcode signals were displayed on several seven-segment displays.

Line Following Robot

An autonomous line following robot programmed in pure HCS12 Assembly

The hardware is a 4-wheeled autonomous robot equipped with five light sensors to detect and follow a line. Front and back bumpers are used to detect dead ends and initiate a U-turn.

The robot can solve a short maze using a left‑turn algorithm at each intersection
